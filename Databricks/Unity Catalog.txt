# Data Governance
- Refers to the process of managing the availability, usability, integrity, and security
1. Data Access Control: Control who has access to which data
2. Data Access Audit: Capture and record all accesses to data, who is using, when and how.
3. Data Lineage: Captures upstream sources and downstream consumres.
4. Data Discovery: Ability to search for and discover authorized assets
5. Secure Data Sharing
